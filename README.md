# gc-erp-mcp-server

A lightweight general-contractor ERP whose product is an **MCP server**. It exposes a commitment-based data model, tool methods, and MCP "apps" (UI components) that an MCP client (Claude Desktop, Claude Code, etc.) renders and orchestrates. Dogfood-first: built for Max + Salman GC'ing our own projects (~1–5/year), not SaaS.

See [SPEC.md](./SPEC.md) for the data model, a narrative walkthrough of a job end-to-end, and open questions.
